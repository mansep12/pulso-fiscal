# Glosario editorial

El objetivo del lenguaje es describir datos publicos de forma clara, verificable y neutral.

## Usar

| Termino | Uso recomendado |
| --- | --- |
| gasto inusual | Cuando un monto supera una regla estadistica publicada. |
| posible exceso | Solo como hipotesis tecnica, nunca como acusacion. |
| requiere explicacion | Cuando el dato no tiene respaldo visible o tiene variacion relevante. |
| dato publico | Para enfatizar que la fuente es oficial y accesible. |
| fuente oficial | Siempre enlazar la URL o documento usado. |
| estimacion | Para litros, kilometros, horas de manejo u otras conversiones. |
| comparacion | Para contrastes entre instituciones similares. |
| alerta tecnica | Para resultados generados por reglas reproducibles. |
| rango estimado | Para cualquier calculo dependiente de supuestos. |

## Evitar

| Termino | Motivo |
| --- | --- |
| corrupto | Acusatorio y no verificable sin resolucion oficial. |
| ladron | Difamatorio. |
| se robaron | Afirma delito sin sentencia. |
| aprovechador | Calificacion subjetiva. |
| culpable | Reservado para resoluciones formales. |
| fraude | Usar solo si existe resolucion oficial o documento judicial/administrativo. |
| escandalo | Editorializa el dato. |
| mafia | Difamatorio y vago. |

## Regla base

Si una frase no se puede respaldar con una fuente oficial y una formula publicada, no debe aparecer como afirmacion del sitio.
